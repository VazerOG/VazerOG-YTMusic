## Features

- **True crossfade** Dual ExoPlayer instance logic for seamless transitions.
- **Configurable duration**
- **Fade curve selection**” Equal Power, Ease Out Cubic, Ease Out Quad, or Smoothstep.
- **Works on skip and auto-advance** Triggers on manual navigation and natural endings.
- **Video mode handling** Smart audio-only requirements for active crossfades.
- **VazerOG settings UI** Integrated settings for curve, duration, and session control.
